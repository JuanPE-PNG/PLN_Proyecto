## Descripción

Este proyecto se trata de un juego de texto, del estilo **escape room** con ambientación de fantasía RPG.
El jugador se encuentra atrapado en una torre, en la cual debe explorar las habitaciones de la torre, examinar objetos del entorno, recoger elementos clave y resolver acertijos para lograr escapar... ¡Todo esto mientras  interactua únicamente mediante comandos escritos en español!.

Algunos de los comandos validos son:
```
> tomar la llave
> abrir el cofre con la llave
> ir al norte
> examinar el pergamino
> leer el libro
```
